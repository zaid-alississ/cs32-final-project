About This Project

This project uses Spotify listening data to generate a personalized taste profile. It looks at a user's top 10 artists over the last 6 months then turns that information into a simple summary of their listening style with recommendations.

How it works

  1. The user logs in with Spotify.
  2. The app collects top artists.
  3. The project computes simple listening scores.
  4. The user receives a taste profile based on those results.
