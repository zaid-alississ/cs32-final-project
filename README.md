# cs32-final-project
Final project with Khalid and Kim

We will build a model that takes data from Letterboxd, such as watched films, ratings, and preferences, and uses it to recommend movies someone would be likely to enjoy. By analyzing patterns in viewing history, the model will create tailored suggestions and a personalized taste profile, in order to make discovering new movies easier and save time for when you don't know what movie to watch.
